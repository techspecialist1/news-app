## Technologies used
* React Native
* Typescript

### How to run the app
* Run these Commands before installing app:

#### With Yarn:

``` yarn && cd ios && pod install && cd .. ```

#### With npm:

``` npm i && cd ios && pod install && cd .. ```

* Installation commands:
### For android:

#### With Yarn: 

``` yarn android ```

#### With npm:

``` npm run android ```

### For ios:

#### With Yarn: 

``` yarn ios ```

#### With npm:

``` npm run ios ```

If metro server is not started, please run ```npx react-native start```

#### Now you are all set!
